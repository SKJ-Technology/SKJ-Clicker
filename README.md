# SKJ Clicker

A powerful and lightweight autoclicker with advanced macro functionality, developed by **SKJ Tech** (Sklep Kuby i Jasia).

## Features

* **High-Speed Autoclicking:** Fast and reliable clicking with customizable intervals.
* **Advanced Macros:** Record, edit, and play back complex mouse and keyboard sequences.
* **User-Friendly Interface:** Simple, intuitive, and clean design.
* **Hotkeys:** Easily start and stop clicking or macro execution using keyboard shortcuts.

## Installation

1. Go to the [Releases](../../releases) page.
2. Download the latest version of `SKJ-Clicker`.
3. Run the application (no installation required).
